# 1. プロジェクト概要

本アプリは、現代のSNS疲れを癒やす「デジタル・サンクチュアリ」である。ユーザーがどんな些細なことでも投稿すれば、AIが圧倒的な速度と熱量で肯定し、称賛する。敵対的な意見（アンチ）を仮想フォロワーが即座に論破する「集団的防衛」の演出により、ユーザーに究極の心理的安全性を体験させる。

# 2. ユーザー体験（UX）設計

- **即時性の充足**: 投稿ボタンを押すとバックエンドがAIリプライを一括生成し、レスポンスとして直接返却。1件目のリプライは即時表示、以降は5〜15秒間隔で順次表示される。
- **万能感の演出**: リプライ表示と連動してリアルタイムに増え続ける「いいね」の数字により、世界の中心にいる感覚を醸成する。
- **ドラマチックな保護**: アンチという「スパイス」をあえて投入し、それを周囲が即座に否定することで、ユーザーとフォロワーの絆を疑似的に強化する（カタルシス効果）。
- **初回オンボーディング**: "普通のSNS"の冷たさ（いいね3件、「で？」の一言）を体験させた後、ZEN-KOTEIの圧倒的な承認体験（Rank 10相当の爆発的リプライ）との対比で世界観に引き込む。白基調の「普通のSNS」画面から、黒×ピンクのZEN-KOTEIへの鮮烈なテーマ切り替えにより、視覚的にもインパクトを与える。
