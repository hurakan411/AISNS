# 5. UI/UX デザイン要件

## 5.1 デザイン言語
- **ベースカラー**: `#010102` (Deep Black)。漆黒の背景がコンテンツとネオンカラーを際立たせる。
- **プライマリカラー**: `#ec4899` (Hot Pink)。「承認」と「愛」を象徴。
- **セカンダリカラー**: `#22d3ee` (Cyan)。「テクノロジー」と「先進性」を象徴。
- **オンボーディング用カラー**: `rgb(246, 246, 249)` (Light Gray-White)。Phase 1〜2で使用する「普通のSNS」演出用の白基調テーマ。本編の黒×ピンクとのコントラストで世界観の転換を強調する。

## 5.2 コンポーネント・スタイリング
- **カード**: 半透明の黒背景に角丸(28pt)を適用し、微かな白ボーダーで囲む。
- **ネオンシャドウ**: 重要な数値（フォロワー数、ランク）やアクセントアイコンには、強めのピンクやシアンのシャドウを適用し、サイバー感（ネオン）を強調する。
- **アバターとタグ**:
  - アバターには `picsum.photos` のランダム画像を使用（通常100種類 + アンチ用20種類）。同一 `author_name` には同じアバターを一貫して使用。
  - アンチは「赤枠・HATER タグ・⚠️アイコン」で強調。
  - 防衛者は「緑枠・GUARDIAN タグ・✅アイコン」で視覚的に差別化。
- **グラスモーフィズム**: ナビゲーションバー等には `UltraThinMaterial` (すりガラス) を適用し、背面が透けて見えるリッチなUIとする。
- **個人情報警告**: 投稿画面（通常・オンボーディング共通）の下部に、⚠️アイコン付きの注意テキストを常時表示。

## 5.3 アニメーション
- **Slide & Fade**: すべての新規要素（投稿・リプライ、画面遷移）はなめらかなトランジションアニメーションで表示。リプライはSpring Animation（response: 0.4, damping: 0.7）で挿入。
- **Pulse**: ステータスバーのランプや「いいね」アイコンを脈動させ、システムが生きていることを強調。
- **Lottie アニメーション**:
  - **Loading Pink Dots**: 返信生成中（pendingRepliesが残っている間）に、返信欄の最上部に中央配置で表示。サイズ: height 70 / maxWidth 140。
  - **Technology isometric AI robot brain**: プロフィール画面のヘッダー背景に表示。indigo/purpleグラデーション背景上に、下寄り配置（height 120, opacity 0.4）。`.clipped()` で上部はみ出しを防止。
- **オンボーディングアニメーション**:
  - Phase間のトランジション: `easeInOut(duration: 0.6)` で自然に切り替え。
  - Ritualテキスト: 各行を0.6秒のフェードインで順次表示。
  - ZEN-KOTEIロゴ: Spring Animation + ネオンシャドウで登場。
  - 「ZEN-KOTEIを始める」ボタン: bottom → up のスライド + フェードで出現。

## 5.4 ランクガイドUI
Stats画面内の全10ランク一覧表示のデザインルール:
- **現在ランク**: Hot Pinkの背景ハイライト + 「NOW」バッジ。全詳細データ可視。
- **次のランク**: Cyanのボーダー + 「NEXT」バッジ。全詳細データ可視。
- **過去のランク**: 通常表示。全詳細データ可視。
- **2つ以上先のランク**: テキスト・アイコンをグレーアウト。ランク名は「？？？」、フォロワー数・リプライ数・いいね数・アンチ数はすべて「???」に置換。
