# 5. UI/UX デザイン要件

## 5.1 デザイン言語
- **ベースカラー**: `#010102` (Deep Black)。漆黒の背景がコンテンツとネオンカラーを際立たせる。
- **プライマリカラー**: `#ec4899` (Hot Pink)。「承認」と「愛」を象徴。
- **セカンダリカラー**: `#22d3ee` (Cyan)。「テクノロジー」と「先進性」を象徴。

## 5.2 コンポーネント・スタイリング
- **カード**: 半透明の黒背景に角丸を適用し、微かな白ボーダーで囲む。
- **ネオンシャドウ**: 重要な数値（フォロワー数、ランク）やアクセントアイコンには、強めのピンクやシアンのシャドウを適用し、サイバー感（ネオン）を強調する。
- **アバターとタグ**: アバターにはリアルな人物画像等を使用する。またアンチは「赤枠・警告アイコン」、防衛者は「緑色の GUARDIAN タグ」をつけ、敵味方を視覚的に分かりやすく差別化する。
- **グラスモーフィズム**: ナビゲーションバー等には `UltraThinMaterial` (すりガラス) を適用し、背面が透けて見えるリッチなUIとする。

## 5.3 アニメーション
- **Slide & Fade**: すべての新規要素（投稿・リプライ、画面遷移）はなめらかなトランジションアニメーションで表示。
- **Pulse**: ステータスバーのランプや「いいね」アイコンを脈動させ、システムが生きていることを強調。
